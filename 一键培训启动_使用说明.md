# 一键培训启动.bat 使用说明

## 1. 文件用途

`一键培训启动.bat` 用于在培训现场快速完成以下动作：

1. 自动打开培训相关文档（安装、使用、配置、飞书打通）
2. 检查 Node.js / npm 环境
3. 检查并安装 `openclaw`
4. 安装 Feishu 插件
5. 调用联调自检脚本
6. 输出标准实操命令，便于讲师现场演示

---

## 2. 使用方法

在 `d:\FY003` 目录双击运行，或在命令行执行：

```bat
d:\FY003\一键培训启动.bat
```

---

## 3. 运行前要求

- Windows 系统可运行 `cmd` / `PowerShell`
- 网络可访问 npm
- 建议提前准备飞书应用 `App ID` / `App Secret`

---

## 4. 培训现场推荐流程

1. 运行 `一键培训启动.bat`
2. 按自动打开的文档讲解“安装 / 使用 / 配置 / 联调”
3. 执行 `openclaw channels add` 完成飞书渠道配置
4. 在飞书发送测试消息，完成 pairing 与回包验证
5. 依据验收清单逐项打勾

---

## 5. 常见问题

### 5.1 Node.js 版本过低

脚本会直接提示并退出，请升级到 `>=22` 后重试。

### 5.2 openclaw 或插件安装失败

通常是网络或 npm 源问题。修复网络后手动执行：

```powershell
npm install -g openclaw@latest
openclaw plugins install @openclaw/feishu
```

### 5.3 飞书仍未打通

优先检查：

1. 飞书应用是否发布
2. 事件订阅是否添加 `im.message.receive_v1`
3. 是否完成 `openclaw pairing approve feishu <配对码>`
4. 查看 `openclaw logs --follow` 定位错误
