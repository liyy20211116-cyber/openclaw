# OpenClaw 安装环境要求

## 1. 适用范围

本要求用于 Windows 培训与日常部署，确保安装与联调过程稳定。

## 2. 必备环境

- 操作系统：Windows 10/11
- 终端：PowerShell
- Node.js：>= 22
- npm：建议 >= 10（随 Node.js 安装）
- 网络：可访问 npm 与飞书开放平台

## 3. 版本检查命令

```powershell
node -v
npm -v
```

## 4. 判定标准

- `node -v` 主版本号 >= 22
- `npm -v` 可正常返回版本号
- 命令无报错

## 5. 常见问题

### 5.1 未安装 Node.js

- 现象：`node` 命令不存在
- 处理：安装官方 Node.js（22.x 或更高）

### 5.2 版本过低

- 现象：脚本提示 `Node.js >= 22 required`
- 处理：升级 Node.js 后重新执行检查命令

### 5.3 网络受限

- 现象：安装包下载失败
- 处理：检查网络、代理或 npm 镜像源
