# 飞书接入配置与联调

## 1. 飞书侧配置步骤

1. 在飞书开放平台创建企业自建应用
2. 开启机器人能力
3. 获取 `App ID` 与 `App Secret`
4. 配置事件订阅：`im.message.receive_v1`
5. 发布应用版本

## 2. OpenClaw 侧接入步骤

```powershell
openclaw plugins install @openclaw/feishu
openclaw channels add
openclaw gateway restart
openclaw gateway status
```

## 3. 联调步骤（必须执行）

```powershell
openclaw logs --follow
```

在飞书私聊机器人发送测试消息。若收到配对码，执行：

```powershell
openclaw pairing list feishu
openclaw pairing approve feishu <配对码>
```

再次发送消息，确认可回复。

## 4. 群聊联调（可选）

- 在群里 @机器人发送测试消息
- 确认群聊中可正常回复

## 5. 常见问题排查顺序

### 5.1 收不到消息

1. 应用是否已发布
2. 是否已添加 `im.message.receive_v1`
3. 网关是否运行中
4. 查看日志是否有权限或鉴权报错

### 5.2 收到消息但不回复

1. 是否已完成 pairing 批准
2. 群聊是否正确 @ 机器人
3. 渠道策略是否限制发送者

### 5.3 凭据泄露

1. 飞书侧立即重置 Secret
2. 更新 OpenClaw 配置
3. 重启网关并重新联调

## 6. 完成标准

- [ ] 私聊可正常收发
- [ ] 配对流程可通过
- [ ] （可选）群聊 @ 机器人可回复
